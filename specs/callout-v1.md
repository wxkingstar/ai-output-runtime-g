# aio:callout@1

Use `aio:callout@1` for conclusions, recommendations, warnings, and risk notes.

Required fields:

- `title`: plain text.
- `body`: plain text with the limited inline Markdown subset.

Optional fields:

- `tone`: `info`, `success`, `warning`, or `danger`. Defaults to `info`.
- `items`: up to 8 plain-text bullet items.

HTML, CSS, and JavaScript are not allowed.
