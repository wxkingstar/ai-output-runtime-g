# Security Policy

AIO renders AI-authored data. Treat all AIO content as untrusted input.

## Supported Version

| Version | Supported |
|---|---|
| 0.1.x | Yes |

## Security Boundary

AIO v0.1 does not support:

- AI-generated HTML.
- AI-generated CSS.
- AI-generated JavaScript.
- Inline event handlers.
- `iframe`, `object`, `embed`, or form submission.
- `javascript:`, `data:`, or `file:` URLs.
- Dynamic schema loading.
- Custom components.

Report security issues through GitHub Security Advisories when the repository is public.
