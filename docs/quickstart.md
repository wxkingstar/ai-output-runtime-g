# Quickstart

## Install Skill

```bash
npx skills add wxkingstar/ai-output-runtime-g -y
```

## Ask Your Agent

```text
按 AIO v0.1 输出一份方案评审。
```

## Validate Output

```bash
node scripts/aio.mjs validate report.md
```

## Render Output

```bash
node scripts/aio.mjs render report.md --out report.html
```

## Browser Runtime

```html
<script src="https://cdn.jsdelivr.net/gh/wxkingstar/ai-output-runtime-g@v0.1.0/assets/ai-output-runtime.js"></script>
```
